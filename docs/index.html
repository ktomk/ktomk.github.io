• Run .travis.yml Github Action :: Run `.travis.yml` build jobs/scripts as a Github Action. Travis-CI is harshly reducing the options for what they call "Open Source Projects" while the .org and .com homepage still announces "Testing your open source projects will always be free!" but many, many projects report they don't offer any practical options for them any longer. No complaints here, this Github Action allows to bring build times on Travis-CI down quickly and to migrate those build jobs easily of which Microsofts Github is able to run. Diversify and maintain flexibility.
